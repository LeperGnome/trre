# trre

Interactive directory viewver. Move around with vim key bindings or arrow keys, copy files with `y` and `p`.

```
> ./.git/index

│  src/
│  .git/
│  │  FETCH_HEAD
│  │  logs/
│  │  │  HEAD
│  │  │  refs/
│  │  ORIG_HEAD
│  │  index
│  │  HEAD
│  │  COMMIT_EDITMSG
│  ▼  description
│  Cargo.toml
│  perf.data
│  .gitignore
│  Cargo.lock
│  target/
```
